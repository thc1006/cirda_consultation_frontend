| 內容摘述 | （500字以內，若有額外補充，請以附件提供，格式不拘） 在數位化與高壓社會環境中，心理健康日益受到重視。尤其大學生與職場族群所面臨之學業、人際與職場壓力以成慣見議題。隨2025年生成式AI技術著步應用於情緒支持與壓力支持場景，然多數模型建立仍倚賴英語或高資源語言資料，對於文化脈絡敏感的華語「壓力表達對話」，支援明顯不足。華語心理健康語料屬於典型「低資源語言資料」，不僅資料量稀少，亦缺乏足夠專家參與，難以透過傳統微調（FT）或人為標註（RLHF）進行大規模優化。因此，探索低資源語境下語言模型的強化方法，具備高度學術價值及社會應用潛力。 因此，本計畫以「低資源語言模型優化」為核心，建立資料合成、擴增與模型訓練／優化流程，並於國網HPC/AI平台部署原型服務，以大學生與職場族群之中文壓力表達為應用測試模型，提供未來於教育與企業心理輔導等領域之應用示範，貢獻「AI for Social Good」。 研究將配合本人國科會醫療人文跨領域計畫，結合學生碩士論文，進行語料的搜集和整理。另外，將邀請心理專家，協助進行語意的辨識，並根據現有模型訓練/微調語料集。完成後，語料集訓練結果可上傳到 RAP平台供展示、測試，但是語料的搜集與整理過程，以及應用與專業討論則待整個計劃和論文完成後，再行公開。 |
| 執行方法與重點項目 | 1\. 運用 PEFT 技術（如 LoRA / Adapter），參數高效微調（PEFT, Parameter-Efficient Fine-Tuning），提升小樣本場域下模型適應能力。 2.混合資料訓練（Hybrid Data Training）:結合有限真實語料與學術文獻生成資料，設計混合訓練策略，提升模型泛化能力。 3.連鎖式思考（COT, Chain-of-Thought）與生成式資料擴增（Synthetic Dialogue Generation）應用 COT 技法於壓力對話語境之資料標註及生成，透過多輪生成優化資料品質，減少專家標註負擔。 4.運用NCHC RAP實作訓練管線與服務部署。 5\. 建置 NYCU 客製化專屬 RAG 受控知識庫，對齊教職員生的壓力源脈絡與校內資源/情境，並以「先檢索、後生成」的引用式回覆提供可溯源建議。 6\. Prompt engineering: CBT/DBT「技能模組化（System Prompt）」與結構化輸出（Skill Library \+ Schema） 7\. 危機偵測與安全分流：自傷/自殺語句的危機模式與資源導引 8\. 量化/質性評估設計：以對話層級 rubric 驗證 CBT/DBT 9\. Red-teaming 與提示詞穩健性：對抗 prompt injection / 角色破壞  |
